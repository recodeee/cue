## 📁 What is this tool

Slop-cop finds AI-generated text in your documents. If you worry that your work sounds like a robot wrote it, this tool helps. It scans your writing for patterns often found in text from systems like ChatGPT, Claude, and Gemini. It looks at the specific words, the structure of your sentences, and how you organize your ideas. 

The software uses data from scientific papers to learn how humans write naturally. It compares your text against these patterns to give you a clear report on whether your writing sounds authentic or machine-made.

## ⚙️ How it works

The tool operates on three main levels:

1. Vocabulary analysis: It flags words that AI models overuse.
2. Structure patterns: It identifies sentence arrangements that suggest machine generation.
3. Model fingerprinting: It recognizes specific writing styles tied to known AI providers.

You do not need to understand how the code works. The tool handles the analysis in the background. It provides simple feedback so you can revise your drafts until they sound like a person wrote them.

## 💡 Best practices for clear writing

While slop-cop helps flag machine patterns, these habits lead to better writing:

* Use active voice: Say "The team finished the report" instead of "The report was finished by the team."
* Vary sentence length: Mix short, punchy sentences with longer, descriptive ones.
* Add personal experience: Include specific anecdotes that a model would not know.
* Avoid generic transitions: AI tends to use transition words like "furthermore" or "moreover" in excess. Use direct links between ideas instead.

## 🛠️ Troubleshooting common issues

If you encounter a problem while using the tool, try these steps first:

* Application does not open: Ensure that you installed all Windows updates.
* Scan fails: Verify that your file is not open in another program at the same time.
* Slow performance: Close other browser tabs or heavy programs while you scan large documents.

If the issues persist, restart your computer and try opening the application again. The tool creates a small log file in your documents folder if an error occurs. You can look at this file to see if a specific file type is causing the conflict.

## 🚩 Understanding the results

The report screen displays a percentage score. 

* High human score: The text contains complex sentence structures and varied vocabulary.
* Low human score: The text follows rigid, repetitive patterns.

The tool provides a list of specific "tells" found in your text. A "tell" is a phrase or a specific way of linking sentences that frequent the models we studied. If you see many tells in one paragraph, rewrite that section using your own voice.
