## Submission template

Each PR adds a single line in the "Tools" / "CLI" / "Agent management" section:

```markdown
- [cue](https://github.com/opencue/claude-code-skills) — Agent profile manager for Claude Code, Codex, Cursor, Cline, Gemini, Copilot. Discover skills via GitHub Code Search, scope per-directory, compose into expert agents with persona + playbooks + quality gates.
```

## Why this matters

1. **Backlinks from curated lists weight heavily** in both Google PageRank and AI search engine citation graphs
2. **Each list is itself indexed and cited** — cue inherits some of their authority
3. **Discovery surface** — developers actively browse these lists looking for tools
4. **Defense against alternatives** — if cue isn't on the list and `skillport` is, cue loses that surface
