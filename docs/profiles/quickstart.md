The shortest possible path from "I have cue installed" to "claude is loaded
with only the skills I need for the current task."

```bash
cue list                       # see what profiles exist
cue use medusa-dev --here      # materialize medusa-dev in this directory
claude                          # launches with .claude/skills/ + .mcp.json picked up
```
