# Skills Admin

Administrative workflows for contributing to and updating the [agent-skills](https://github.com/promptingcompany/agent-skills) repository.

## Workflows

| Workflow | Triggers |
|---|---|
| Open a PR | "open a PR", "submit my changes", "push this skill", "create a pull request" |
| Update an installed skill | "update my skills", "get the latest skill", "how do I update" |

## Install

```bash
cp -r skills/skills-admin ~/.claude/skills/
```

See [`INSTALL.md`](INSTALL.md) for claude.ai and MCP server setup.
