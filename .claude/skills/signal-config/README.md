# Signal Config

Generate YAML signal configs for agent simulation experiments. Covers signal definition, extraction methods, fold, and aggregation.

## Workflows

| Workflow | Triggers |
|---|---|
| Generate config | "generate a signal config", "create signals for my experiment", "I want to track [metric]", "write a signal YAML" |
